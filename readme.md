# CircleFramework
這是我用ChatGPT開發的一個框架
用於我的專案:D

# 如何使用

首先你得先compile spigot 1.8.8 & 1.21.11
可以用buildtools 但記得讓他存到local repo

然後clone這個 之後compile
接著再插件depend寫這個

```xml
<dependency>
    <groupId>top.itsglobally</groupId>
    <artifactId>CircleFramework</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```

就可以了
